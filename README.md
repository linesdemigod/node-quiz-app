# A simple quiz application based on node js and react js

Rename **_.env.example_** to **_.env_**

### Install Dependencies

```
cd backend
```

```
cd frontend
```

```
npm install
```

### Run Migration

```
backend
node models/index.js
```

### Start server and tailwind css

```
cd backend
npm run start
```

```
cd frontend
npm run dev
```
